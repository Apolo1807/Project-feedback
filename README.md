
# Project Feedback

Project made to train my skills in API's using Spring boot and Angular



## Tech Stack

**Client:** Angular

**Server:** Spring Boot

## Versions

```
Angular 11
```
```
Java 7
```

## Run 

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
